# Project Title

"Predicting Employee Layoff Risk in the Age of AI using Machine Learning"

# Problem Statement

Companies adopting AI want to understand which employee profiles are most vulnerable to automation and layoffs. By analyzing workforce characteristics and AI adoption metrics, we can identify the major factors influencing layoff risk.
